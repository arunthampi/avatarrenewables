Power Tariff Alert

This is a state of the unique product from AVATAR Renewables, designed for a large number of ENERGY MANAGEMENT and AUDIT applications.

This microcontroller based equipment has the below in-built functions. The device will measure electrical parameters on a 3 phase 4 wire AC circuit. It is a 3-in-1 system that does POWER-MEASUREMENT, DATA-LOGGING and WIRELESS REMOTE DATA TRANSMISSION.

The system has distinct and powerful features and can find extensive use in:-

LARGE BUILDINGS, OFFICES, APARTMENT COMPLEXES

SOLAR ENERGY/GRID SYSTEMS

POWER AUDITING (records data with time stamp locally)

POWER-ENERGY RESEARCH/ANALYSIS

GENERATOR SET MONITORING

UTILITY/POWER PLANTS
